# FlowChain

**The Oracle in Your Ear.**

FlowChain is a voice-native, autonomous "Guardian Agent" for the Neo N3 Blockchain, secured by SpoonOS Trusted Execution Environments. It protects your assets and keeps you informed with personalized audio intelligence.

## Features
- **Voice-Native Interface**: Interact with your crypto portfolio using natural language.
- **Secure Execution**: Runs inside a TEE for institutional-grade security.
- **Neo N3 Integration**: Native support for GAS/NEO transfers and swaps.

## Installation

### Prerequisites
- Python 3.10+
- SpoonOS CLI
- Neo N3 Wallet

### Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure environment variables in `.env` (see `src/config.py` for required keys).
4. Run the agent:
   ```bash
   python src/main.py
   ```
